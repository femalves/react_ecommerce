### :point_up: Frontend

```bash

# Clone this repository
$ git clone https://github.com/femalves/react_ecommerce

# From the terminal, go to folder
$ cd silverline-jewelry/client

# Install dependencies
$ npm install

# Run in development mode
$ npm start

# Go to http://localhost:3000

```
